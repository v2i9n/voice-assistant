# 🎙️ Voice Assistant - مساعد صوتي

مساعدك الشخصي يرد عليك بالصوت وينفذ أوامرك

### 🎯 الأوامر المتاحة
- "كم الساعة" - يقول الوقت
- "افتح يوتيوب" - يفتح YouTube
- "ابحث عن..." - يبحث في قوقل
- "اقفل" - يقفل البرنامج
